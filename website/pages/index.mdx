## Quick Start

<div className="steps-container mt-8">

### Step 1: Use This Template

```bash
# Clone for your startup
npx degit pamasiga/pamasiga-os my-startup
cd my-startup
```

### Step 2: Run Setup Wizard

```bash
pnpm install
pnpm run setup
```

The wizard configures your project name, branding, GitHub details, and team information.

### Step 3: Create Repository

```bash
git init
git add .
git commit -m "init: setup from PAMASIGA OS template"
gh repo create my-startup --public --source=. --push
```

### Step 4: Start Building

```bash
# Start docs site
cd website && pnpm dev

# Add your code to:
# - apps/ (frontend, mobile)
# - services/ (backend APIs)
# - packages/ (shared code)
```

</div>
